## 기능

- [x] 인기 영화 데이터 조회
  - 영화 포스터 이미지
  - hover 시 제목, 내용요약, 평점 정보 확인 가능
- [x] 영화 검색 기능
  - 제목 기반
  - 검색어 입력 완료 시 1초 후에 검색 시작
  - 띄어쓰기 무시 _(ex. 쇼생크탈출 => 쇼생크 탈출도 검색 가능)_
  - 검색 결과 없을 시 안내
- [x] 스크롤 감지 헤더
  - 스크롤을 내리면, 헤더가 올라가서 보이지 않게 됨
  - 스크롤을 올리면, 헤더가 다시 내려옴

## 기능

- [x] 인기 영화 데이터 조회
  - 영화 포스터 이미지
  - hover 시 제목, 내용요약, 평점 정보 확인 가능
- [x] 영화 검색 기능
  - 웹사이트 로드 시 입력창에 자동 포커스
  - 제목 기반 검색
  - 검색 버튼 클릭 or 엔터 시 검색
  - 대소문자, 띄어쓰기 관계없이 검색
  - 검색 결과 없을 시 안내
- [x] 스크롤 감지 헤더
  - 스크롤을 내리면, 헤더가 올라가서 보이지 않게 됨
  - 스크롤을 올리면, 헤더가 다시 내려옴
- [x] 다국어 지원 (영어, 한국어)

- [x] 반응형 페이지
